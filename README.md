# Real-Time Traffic Prediction and Analysis

## Description
This project predicts traffic congestion and travel time using machine learning techniques.

## Features
- Traffic volume prediction
- Weather and AQI based analysis
- Route visualization using OpenStreetMap

## Technologies Used
- Python
- Pandas, NumPy, Scikit-learn
- Matplotlib, Seaborn

## How to Run
1. Clone the repository
2. Install dependencies
3. Run the Jupyter Notebook
